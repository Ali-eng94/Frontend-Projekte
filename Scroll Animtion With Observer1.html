Wie funktionieren diese Animationen?

Diese Seite verwendet CSS Scroll-Driven Animations - eine neue CSS-Spezifikation, die es ermöglicht, Animationen direkt mit dem Scrollfortschritt zu verknüpfen.

Vorteile von CSS-Only Animationen

  • Kein JavaScript notwendig
  • Bessere Performance
  • Glattere Animationen
  • Einfachere Wartung
  • Nativ vom Browser gerendert

Browser-Unterstützung

CSS Scroll-Driven Animations werden von modernen Browsern unterstützt (Chrome, Edge, Safari). Für ältere Browser gibt es einen Fallback.
